# 🔐 Password Generator

A simple and secure **Password Generator** built with **Python**. This project generates strong random passwords based on the user's preferred length, helping improve online security with unique and hard-to-guess passwords.

---

## 🚀 Features

* Generate secure random passwords
* User-defined password length
* Fast and lightweight
* Beginner-friendly Python project
* Clean and easy-to-understand code

---

## 🛠️ Technologies Used

* Python 3
* Random module

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/jrtiger2005/Password-Generator.git
```

2. Navigate to the project folder:

```bash
cd Password-Generator
```

3. Run the program:

```bash
python password_generator.py
```

Strong passwords help protect your online accounts from brute-force attacks and unauthorized access. Using randomly generated passwords with letters, numbers, and special characters significantly improves security.
